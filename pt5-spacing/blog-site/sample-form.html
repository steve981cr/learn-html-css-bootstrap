Box Model Example

In CSS you can use the following length units:

Absolute units:
• px - pixels - 1/96th of 1 inch. Equals 1 pixel dot displayed on low dpi devices.
Relative units:
• % - percent relative to the parent size.
• em - size in relation to the current element.
• rem - size in relation to the root element (16px)
